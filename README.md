# Game hit the number RUBY
> Game made in ruby ​​for Terminal, and educational purposes

## Running locally

1. Clone this repo:

```sh
$ git clone https://github.com/dinizdev/MiniGameRuby.git
```

2. Then go to the project's folder:

```sh
cd MiniGameRuby
```

3. Install all dependencies:

```sh
gem install
```

4. Run locally:

```sh
ruby Jogo.rb
```
